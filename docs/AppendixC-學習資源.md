## C.7 套件管理

### pip（Python Package Installer）

```bash
# 安裝套件
pip install package_name

# 安裝特定版本
pip install package_name==1.2.3

# 升級套件
pip install --upgrade package_name

# 列出已安裝
pip list

# 匯出需求檔
pip freeze > requirements.txt

# 從需求檔安裝
pip install -r requirements.txt
```

### 虛擬環境

```bash
# 建立虛擬環境
python -m venv myenv

# 啟動（Windows）
myenv\Scripts\activate

# 啟動（macOS/Linux）
source myenv/bin/activate

# 停用
deactivate
```

## C.11 學習路徑建議

### 初學者路徑（0-3 個月）

1. **第 1 個月：基礎語法**
   - 變數、資料型別、運算符
   - 條件判斷與迴圈
   - 列表、字典基本操作
   - 完成本教程 Chapter 1-3

2. **第 2 個月：進階語法**
   - 函數與模組
   - 檔案操作
   - 錯誤處理
   - 完成本教程 Chapter 4-7

3. **第 3 個月：實戰練習**
   - 獨立完成 3-5 個小專案
   - LeetCode 簡單題 20 題
   - 閱讀他人程式碼

### 中級工程師路徑（3-12 個月）

1. **深入 Python**
   - 物件導向程式設計
   - 裝飾器、生成器
   - 多執行緒與並行

2. **選擇專精領域**
   - Web 開發：Django/Flask
   - 資料科學：Pandas/NumPy
   - 自動化：Selenium/系統腳本

3. **參與開源專案**
   - 在 GitHub 上貢獻代碼
   - 閱讀優質開源專案
